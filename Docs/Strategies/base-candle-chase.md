# 10min Base Candle Chase

## 역할

10분봉에서 거래량과 거래대금이 터진 기준양봉 후보를 찾고, 현재가가 기준봉 고가를 돌파할 때 추격 후보로 분류한다.

## 시장 범위

- 범위: SOR
- 기준가: KRX 전일종가 고정
- NXT 현재가를 보더라도 기준가를 NXT 종가로 바꾸지 않는다.

## 기본 판단

- 완성된 10분봉 기준
- 양봉
- 거래량 증가
- 거래대금 증가
- 종가가 봉 상단부
- 고가 돌파 후 과열 구간이 너무 멀지 않을 것

## 손절과 목표

- 손절: 기준봉 중심선 또는 돌파가격 아래 제한폭
- 1차 목표: 기준고가 + 기준폭 0.5배
- 2차 목표: 기준고가 + 기준폭 1.0배

## 주의

즉시 주문 전략이 아니라 후보 판정 슬롯이다. 실제 주문은 별도 리스크 가드가 판단한다.
