# Theme / Disclosure Assist

## 역할

뉴스, 공시, 테마 태그를 이용해 후보의 우선순위를 보조한다.

## 시장 범위

- 범위: 보조
- 단독 매수신호 없음
- 매매 판단보다 낮은 우선순위

## 기본 판단

- 좋은 공시 키워드
- 당일 뉴스 재료
- 테마 연결성
- 종목명과 재료의 관련성

## 적용 방식

이 슬롯은 점수나 태그를 올릴 수 있지만 단독으로 주문을 만들지 않는다.

## 주의

키움 실시간 데이터와 주문 판단을 방해하면 안 된다. 뉴스/공시 API 제한이 감지되면 쉬는 쪽을 우선한다.
