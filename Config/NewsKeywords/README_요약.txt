핵심 방향:
뉴스 단어는 코드에 직접 넣지 말고 외부 JSON 파일에서 불러온다.
처음 목적은 뉴스판 태그 표시다.
자동 매수 신호로 바로 쓰지 않는다.
제목 단어는 강하게, 본문 단어는 보조로 본다.
긍정 단어와 주의/부정 단어가 같이 있으면 주의/부정을 함께 표시한다.
